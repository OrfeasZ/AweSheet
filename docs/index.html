Documentation

Creating a new Workbook

To create a new workbook you can can either click on the New button under the File menu, or press the Ctrl+N shortcut.

If you already have another active workbook you will be presented with the following options:

  • Save and Continue, which when selected saves the Workbook and creates a new one.
  • Discard Changes, which when selected discards any pending changes and creates a new workbook.
  • Or Cancel, which when selected cancels the action.

Saving your Workbook

To save the workbook you're currently editing you can either click on the Save or Save As buttons under the File menu, or use the Ctrl+S and Ctrl+Shift+S shortcuts.

If the workbook you're editing has not been saved before you will be prompted to specify a storage location.

Opening a Workbook

To open a previously saved workbook you can either click on the Open button under the File menu, or use the Ctrl+O shortcut.

You will then be prompted with a dialog where you will have to select the workbook you wish to open.

Creating and Deleting Sheets

To create a sheet simply click on the + icon present at the bottom tab navigation.

To delete a sheet simply hover over it, and click on the x button that will appear.

Note: You cannot delete the currently edited sheet.

Exporting a Sheet

AweSheet allows you to easily export your currently active sheet as a CSV file in order to allow cross-compatibility with other applications.

To export your sheet, simply click on the Export button under the File menu, and select a destination file for the export.

Importing a Sheet

AweSheet allows you to easily import sheets from CSV files in order to allow cross-compatibility with other applications.

To import a sheet, simply click on the Import button under the File menu, and select a CSV file of your choice.

Note: Importing a CSV file will overwrite your currently active sheet.

Editing a Cell

To edit a cell you can double click on it, select it and start typing, or select it and type on the text box present in the top toolbar.

After you've finished your entry you can confirm the input by hitting ENTER (or the check button in the top toolbar), or revert to the prevous value by hitting ESC (or the cancel button in the top toolbar).

Inserting Functions

In AweSheet you can insert functions in two different ways: by typing the function directly, or by using the function insertion dialog.

To add a function using the insertion dialog, simply select a cell and press the lightning button in the top toolbar. You can then follow the on-screen details in order to fill in the required fields for the function you're trying to insert. Alternatively, you can insert a function manually when editing a cell.

All functions in AweSheet follow the following pattern:

=function(parameter1, parameters2, ...)

Where function is the function name, and parameterN is one of the following:

  • A cell (ie. A2).
  • Another function.
  • A generic value.

Creating Charts

AweSheet supports the creation of two types of charts: Bar charts, and Line Charts.

To create a chart, you will first need to select the cells that contain the data for the given chart.

Data should be formatted in the following manner:


[         ] [ entry 1 ] [ entry 2 ]
[ label 1 ] [ value 1 ] [ value 1 ]
[ label 2 ] [ value 2 ] [ value 2 ]
[ label 3 ] [ value 3 ] [ value 3 ]

Afterwards, you can select the type of chart you want to create by clicking on either Create Bar Chart or Create Line Chart under the Chart menu.

You will then be prompted to fill in some extra (optional) details for your chart. Once you're satisfied with your input, you can click on Create Chart to generate the chart.
